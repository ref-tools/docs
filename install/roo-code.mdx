title: "Roo Code"
description: "Connect Ref to Roo Code assistants."
---

## Setup

1. Edit the Roo Code MCP servers list.
2. Paste the Ref HTTP configuration.

```json
{
  "Ref": {
    "type": "http",
    "url": "https://api.ref.tools/mcp?apiKey=YOUR_API_KEY"
  }
}
```

## Check

- Restart Roo Code and send a docs query.
- Confirm Ref responds without delay.
