title: "Windsurf"
description: "Connect Ref to the Windsurf agent."
---

## Install

1. Open Windsurf settings.
2. Add the Ref HTTP endpoint.

```json
{
  "Ref": {
    "type": "http",
    "url": "https://api.ref.tools/mcp?apiKey=YOUR_API_KEY"
  }
}
```

## Test

- Ask Windsurf to search docs for a service you indexed.
- Confirm the transcript shows the Ref tool response.
