## Guide the organization

You need to help a team of engineers operate with high throughput using AI while maintaining coherence towards company objectives. You use Ref Plans as a basis for organizational process around plan review.

**Your workflow:**
- Review plans before code is written — catch architecture issues early
- Use the [MCP server](/plans/install/index) to have your agent review active and recently finished plans to understand the team's state
- Set up [team visibility](/plans/collaboration/team-visibility) to see all active plans in one place
