## What is Ref?

Ref is a documentation search and retrieval service designed specifically for AI agents and software engineering tools. It provides semantic search, web search, and URL reading capabilities that enable AI tools to access relevant documentation quickly and accurately.

## How It Works

Ref operates through two main interfaces:

1. **REST API** - Direct HTTP endpoints for search and URL reading
2. **MCP Server** - Model Context Protocol integration for seamless AI tool connectivity

### Key Features

- **Semantic Search**: Find relevant documentation using natural language queries
- **Web Search**: Access real-time information from the web
- **URL Reading**: Extract and process content from specific URLs
- **Team Workspaces**: Collaborate with shared documentation indices
- **Flexible Pricing**: From free tier to enterprise solutions
