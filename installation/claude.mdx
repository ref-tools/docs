## Overview

Claude Desktop natively supports the Model Context Protocol (MCP), making it easy to integrate Ref's documentation search capabilities directly into your conversations with Claude.

<Info>
This guide covers Claude Desktop integration. For Claude web interface, use the [API integration guide](/installation/api) instead.
</Info>

## Prerequisites

Before starting, ensure you have:

- **Claude Desktop** installed (latest version recommended)
- **Node.js 18+** for running the MCP server
- **Ref API Key** from your [dashboard](https://ref.tools/dashboard)

## Installation Methods

### Method 1: NPM Package (Recommended)

The easiest way to get started is using our published NPM package.

<Steps>
  <Step title="Install the MCP Server">
    ```bash
    npm install -g @ref-tools/mcp-server
    ```
  </Step>

  <Step title="Configure Claude Desktop">
    Open Claude Desktop's configuration file:
    
    **macOS:**
    ```bash
    open ~/Library/Application\ Support/Claude/claude_desktop_config.json
    ```
    
    **Windows:**
    ```bash
    notepad %APPDATA%\Claude\claude_desktop_config.json
    ```
    
    **Linux:**
    ```bash
    nano ~/.config/Claude/claude_desktop_config.json
    ```
  </Step>

  <Step title="Add Ref Configuration">
    Add the following configuration to your `claude_desktop_config.json`:
    
    ```json
    {
      "mcpServers": {
        "ref": {
          "command": "ref-mcp-server",
          "env": {
            "REF_API_KEY": "your-api-key-here"
          }
        }
      }
    }
    ```
  </Step>

  <Step title="Restart Claude Desktop">
    Close and restart Claude Desktop for the changes to take effect.
  </Step>
</Steps>

### Method 2: Direct Node.js

For more control, you can run the MCP server directly with Node.js.

<Steps>
  <Step title="Clone or Download">
    ```bash
    git clone https://github.com/ref-tools/ref-tools-mcp.git
    cd ref-tools-mcp
    npm install
    ```
  </Step>

  <Step title="Build the Server">
    ```bash
    npm run build
    ```
  </Step>

  <Step title="Configure Claude Desktop">
    Update your `claude_desktop_config.json`:
    
    ```json
    {
      "mcpServers": {
        "ref": {
          "command": "node",
          "args": ["/path/to/ref-tools-mcp/dist/index.js"],
          "env": {
            "REF_API_KEY": "your-api-key-here"
          }
        }
      }
    }
    ```
  </Step>

  <Step title="Restart Claude Desktop">
    Restart Claude Desktop to load the new configuration.
  </Step>
</Steps>

- Read URL content using `ref_read_url`

## Available Tools

Once installed, Claude Desktop will have access to these Ref tools:

<AccordionGroup>
  <Accordion title="ref_search_documentation">
    **Purpose:** Search through indexed documentation
    
    **Usage:** Claude will automatically use this when you ask questions about documentation
    
    **Example:** "Find React component lifecycle methods"
  </Accordion>

  <Accordion title="ref_read_url">
    **Purpose:** Extract and read content from any URL
    
    **Usage:** Ask Claude to read specific documentation pages or GitHub files
    
    **Example:** "Read the README from https://github.com/facebook/react"
  </Accordion>
</AccordionGroup>

## Configuration Options

### Environment Variables

Customize your Ref integration with environment variables:

```json
{
  "mcpServers": {
    "ref": {
      "command": "ref-mcp-server",
      "env": {
        "REF_API_KEY": "your-api-key-here",
        "REF_API_ENDPOINT": "https://ref.tools/api",
        "REF_TIMEOUT": "30000",
        "REF_MAX_RESULTS": "10"
      }
    }
  }
}
```

### Available Options

| Variable | Default | Description |
|----------|---------|-------------|
| `REF_API_KEY` | Required | Your Ref API key |
| `REF_API_ENDPOINT` | `https://ref.tools/api` | Ref API endpoint |
| `REF_TIMEOUT` | `30000` | Request timeout in milliseconds |
| `REF_MAX_RESULTS` | `10` | Maximum search results to return |

```json
    }
  }
}
```

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="MCP Server Not Starting">
    **Symptoms:** No Ref tools available in Claude Desktop
    
    **Solutions:**
    - Check that Node.js 18+ is installed: `node --version`
    - Verify the configuration file syntax is valid JSON
    - Ensure the API key is correct and has no extra spaces
    - Check Claude Desktop's error logs
  </Accordion>

  <Accordion title="Authentication Errors">
    **Symptoms:** "Invalid API key" or "Unauthorized" errors
    
    **Solutions:**
    - Verify your API key is copied correctly from the dashboard
    - Check that the key hasn't expired
    - Try generating a new API key
    - Ensure no quotes or extra characters in the key
  </Accordion>

  <Accordion title="Search Not Working">
    **Symptoms:** No results or error messages when searching
    
    **Solutions:**
    - Check your credit balance in the dashboard
    - Verify your internet connection
    - Try a different search query
    - Check if the documentation you're searching for is indexed
  </Accordion>

  <Accordion title="Slow Response Times">
    **Symptoms:** Long delays when using Ref tools
    
    **Solutions:**
    - Check your internet connection speed
    - Increase the timeout value in configuration
    - Try searching for more specific terms
    - Contact support if issues persist
  </Accordion>
</AccordionGroup>

### Debug Mode

Enable debug logging to troubleshoot issues:

```json
{
  "mcpServers": {
    "ref": {
      "command": "ref-mcp-server",
      "env": {
        "REF_API_KEY": "your-api-key-here",
        "REF_DEBUG": "true"
      }
    }
  }
}
```

### Logs Location

Check Claude Desktop logs for detailed error information:

- **macOS:** `~/Library/Logs/Claude/`
- **Windows:** `%LOCALAPPDATA%\Claude\Logs\`
- **Linux:** `~/.local/share/Claude/logs/`
