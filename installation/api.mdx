## Quick Start

### 1. Get Your API Key

Copy your API key from the [Ref dashboard](https://ref.tools/dashboard).

### 2. Make Your First Request

<CodeGroup>

```bash cURL
curl -X POST https://ref.tools/api/search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "React hooks tutorial"
  }'
```

```python Python
import requests

headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

response = requests.post(
    "https://ref.tools/api/search",
    headers=headers,
    json={"query": "React hooks tutorial"}
)

data = response.json()
print(data["results"])
```

```javascript Node.js
const axios = require('axios');

const response = await axios.post('https://ref.tools/api/search', {
  query: 'React hooks tutorial'
}, {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

console.log(response.data.results);
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]string{"query": "React hooks tutorial"}
    jsonPayload, _ := json.Marshal(payload)
    
    req, _ := http.NewRequest("POST", "https://ref.tools/api/search", 
        bytes.NewBuffer(jsonPayload))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")
    
    client := &http.Client{}
    resp, _ := client.Do(req)
    
    // Process response
}
```

## Authentication

All API requests require authentication using Bearer tokens:

```bash
Authorization: Bearer ref_api_1234567890abcdef
```

<Warning>
Keep your API key secure:
- Store in environment variables
- Never commit to version control
- Rotate keys regularly
- Use different keys for different environments
</Warning>

## Rate Limiting

The API implements rate limiting to ensure fair usage:

| Plan | Rate Limit | Burst Limit |
|------|------------|-------------|
| **Free** | 100 requests/hour | 10 requests/minute |
| **Basic** | 1,000 requests/hour | 50 requests/minute |
| **Pro** | 5,000 requests/hour | 100 requests/minute |
| **Team** | 10,000 requests/hour | 200 requests/minute |

Rate limit headers are included in responses:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200
```

## Error Handling

The API returns standard HTTP status codes:

<AccordionGroup>
  <Accordion title="200 OK">
    Request successful, results returned
    ```json
    {
      "results": [...],
      "creditsUsed": 1
    }
    ```
  </Accordion>

  <Accordion title="400 Bad Request">
    Invalid request format or parameters
    ```json
    {
      "error": "invalid_request",
      "message": "Query parameter is required",
      "code": "MISSING_QUERY"
    }
    ```
  </Accordion>

  <Accordion title="401 Unauthorized">
    Invalid or missing API key
    ```json
    {
      "error": "unauthorized",
      "message": "Invalid API key",
      "code": "INVALID_API_KEY"
    }
    ```
  </Accordion>

  <Accordion title="429 Too Many Requests">
    Rate limit exceeded
    ```json
    {
      "error": "rate_limit_exceeded",
      "message": "Too many requests",
      "retryAfter": 60
    }
    ```
  </Accordion>

  <Accordion title="402 Payment Required">
    Insufficient credits
    ```json
    {
      "error": "insufficient_credits",
      "message": "Not enough credits for this operation",
      "creditsRequired": 2,
      "creditsAvailable": 0
    }
    ```
  </Accordion>
</AccordionGroup>

## Integration Examples

### Web Application

Build a documentation search interface:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Doc Search</title>
</head>
<body>
    <input id="query" placeholder="Search documentation..." />
    <button onclick="search()">Search</button>
    <div id="results"></div>

    <script>
    async function search() {
        const query = document.getElementById('query').value;
        const response = await fetch('/api/search', {
            method: 'POST',
            headers: {
                'Authorization': 'Bearer YOUR_API_KEY',
                'Content-Type': 'application/json'
            },
            body: JSON.stringify({query})
        });
        
        const data = await response.json();
        displayResults(data.results);
    }
    
    function displayResults(results) {
        const container = document.getElementById('results');
        container.innerHTML = results.map(r => 
            `<div><h3><a href="${r.url}">${r.title}</a></h3><p>${r.excerpt}</p></div>`
        ).join('');
    }
    </script>
</body>
</html>
```

### CLI Tool

Create a command-line documentation search tool:

```python
#!/usr/bin/env python3
import os
import sys
import requests

def search_docs(query):
    response = requests.post(
        'https://ref.tools/api/search',
        headers={
            'Authorization': f'Bearer {os.environ["REF_API_KEY"]}',
            'Content-Type': 'application/json'
        },
        json={'query': query}
    )
    
    if response.status_code == 200:
        results = response.json()['results']
        for result in results:
            print(f"{result['title']}")
            print(f"{result['url']}")
            print(f"{result['excerpt']}\n")
    else:
        print(f"Error: {response.status_code}")

if __name__ == '__main__':
    if len(sys.argv) > 1:
        search_docs(' '.join(sys.argv[1:]))
    else:
        print("Usage: search-docs <query>")
```

### Mobile App Integration

Integrate with React Native:

```javascript
import React, { useState } from 'react';
import { View, TextInput, Button, ScrollView, Text } from 'react-native';

const DocSearch = () => {
  const [query, setQuery] = useState('');
  const [results, setResults] = useState([]);

  const search = async () => {
    const response = await fetch('https://ref.tools/api/search', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({query})
    });
    
    const data = await response.json();
    setResults(data.results);
  };

  return (
    <View>
      <TextInput 
        value={query}
        onChangeText={setQuery}
        placeholder="Search documentation..."
      />
      <Button title="Search" onPress={search} />
      <ScrollView>
        {results.map(result => (
          <View key={result.url}>
            <Text>{result.title}</Text>
            <Text>{result.excerpt}</Text>
          </View>
        ))}
      </ScrollView>
    </View>
  );
};
```

## Security Best Practices

<CardGroup cols={2}>
  <Card title="API Key Management">
    - Store keys in environment variables
    - Use different keys per environment
    - Rotate keys regularly
    - Never log API keys
  </Card>
  
  <Card title="Request Security">
    - Always use HTTPS
    - Validate user input
    - Implement request signing for sensitive apps
    - Use IP allowlisting for server apps
  </Card>
  
  <Card title="Error Handling">
    - Don't expose internal errors
    - Log errors for debugging
    - Provide helpful user messages
    - Implement exponential backoff
  </Card>
  
  <Card title="Performance">
    - Implement client-side caching
    - Use batch requests when possible
    - Monitor and optimize query patterns
    - Set appropriate timeouts
  </Card>
</CardGroup>

Track your API usage:

## Testing

Test your integration thoroughly:

<CodeGroup>

```javascript Jest Tests
describe('Ref API Integration', () => {
  test('should search documentation', async () => {
    const client = new RefClient({apiKey: 'test-key'});
    const results = await client.search('React hooks');
    
    expect(results).toHaveLength(10);

});
```

```python Python Tests
import unittest
from ref_tools import RefClient

class TestRefClient(unittest.TestCase):
    def setUp(self):
        self.client = RefClient(api_key='test-key')
    
    def test_search(self):
        results = self.client.search('React hooks')
        self.assertEqual(len(results), 10)
        self.assertIn('title', results[0])
    
    def test_invalid_key(self):
        client = RefClient(api_key='invalid')
        with self.assertRaises(Exception):
            client.search('test')
```
