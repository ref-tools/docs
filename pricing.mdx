## Credit Usage

Understanding how credits work:

### API Operations

| Operation | Credits | Description |
|-----------|---------|-------------|
| **Search Documentation** | 1 | Semantic search through indexed docs |
| **Read URL** | 1 | Extract content from any URL |
| **Chat** | 1 | Interactive conversations |
| **Web Search** | 2 | Real-time web search results |

### Indexing Operations

| Operation | Credits | Description |
|-----------|---------|-------------|
| **Web Documentation** | 10 | Index a website or documentation site |
| **GitHub Repository** | 0 | Free indexing of public GitHub repos |
| **PDF Files** | 0 | Free PDF document indexing |

<Info>
Indexing operations are one-time costs. Once indexed, searching the content only uses standard search credits.
</Info>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Do credits expire?">
    No, credits never expire. Purchase credits once and use them whenever you need them.
  </Accordion>

  <Accordion title="Can I change plans anytime?">
    Yes, you can upgrade or downgrade your plan at any time. Changes take effect immediately.
  </Accordion>

  <Accordion title="What happens if I run out of credits?">
    API calls will fail when you have insufficient credits. Purchase more credits to continue using Ref.
  </Accordion>

  <Accordion title="Is there a free trial?">
    Every account starts with 200 free credits - no trial needed! This lets you test all features before upgrading.
  </Accordion>

  <Accordion title="Do team members share credits?">
    Yes, team credits are pooled and shared across all team members. Any team member can use credits from the shared pool.
  </Accordion>

  <Accordion title="Can I get a refund?">
    Contact us at hello@ref.tools for refund requests. We'll work with you to resolve any issues.
  </Accordion>
</AccordionGroup>
