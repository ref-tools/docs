## What to Include

- Resource type: public web, GitHub, PDF, or Markdown.
- Access requirements: auth, tokens, or VPN.
- Urgency and launch deadlines.

## Submit

- Email `hello@ref.tools` with the resource list.
- For large repos share the top folders to prioritize.
- Provide any crawl exclusions upfront.

## Tracking

- Support replies with an ETA and credit impact.
- Use the changelog page to see when content lands.
